# xlsx_furigana_delete

## このプラグインについて
QGISでExcel（xlsx）のレイヤのふりがなを削除するプラグイン

## インストール方法
インストール用ZIPファイル「xlsx_furigana_delete_x.x.zip」をダウンロードして、プラグインをインストールしてください。
インストールすると、プラグインツールバーにアイコンが追加されます。

## 使用方法
ふりがな付きで読み込まれたExcel（xlsx）レイヤがある場合に、プラグインツールバーのアイコンをクリックします。  
レイヤ選択ダイアログが表示されるので、レイヤを選択します。  
※レイヤをあらかじめ選択していると、そのレイヤがリストのはじめに表示されます。  
OKボタンを押してプラグインを実行すると、レイヤのもととなっているExcelファイルのふりがなを削除して上書きします。  
同じフォルダにバックアップファイルも作成します。
